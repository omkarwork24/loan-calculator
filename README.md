💎 EMI Calculator

A simple EMI calculator built using HTML, Tailwind CSS, and JavaScript.

🔗 Live Demo: https://omkarwork24.github.io/loan-calculator/

🚀 Features
Clean and modern UI (glass-style design)
Dark / Light mode support
Real-time EMI calculation
Works with sliders and manual input
Handles zero interest cases
Displays values in ₹ (Indian format)
📊 Chart
Shows Principal vs Interest
Updates automatically when values change
📋 Amortization Table
Monthly breakdown:
Principal
Interest
Balance
Scrollable with fixed header
Expand / collapse option
🛠 Tech Used
HTML
Tailwind CSS
JavaScript
Chart.js
